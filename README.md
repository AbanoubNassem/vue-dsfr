# VueDsfr

## Cloner le projet

```shell
git clone https://github.com/laruiss/vue-dsfr
```

## Installer les dépendances

```
npm install
```

### Lancer le storybook

```
npm run storybook
```

### Compiler et minifier pour la production

```
npm run build
```

### Lancer les tests

```
npm run test
```

### Linter les fichiers (CI)

```
npm run lint
```

### Linter et réparer les fichiers

```
npm run format
```
